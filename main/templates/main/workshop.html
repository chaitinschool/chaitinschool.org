Workshop: Fast and fun Django web app development (Oct 2021)

What is it?

This is a workshop on Python and Django. Theodore Keloglou (https://github.com/sirodoht) will show how one can build a very simple web application and how to deploy it using nginx and uWSGI on a Linux cloud instance.

Who is it for?

This workshop is for beginner programmers or programmers who want to see how the Python web world works. This workshop assumes basic programming knowledge.

What will you get out it?

You will see the how you can create and set up a Python Django web application. You will also see how to deploy WSGI-compatible web applications in a simple, straightforward way.

Location

The workshop is taking place at SPACE4,
